Mac 資料庫的 Update 是什麼？可以刪掉嗎？

發表於 2020-12-27 更新於 2021-01-03 分類於 Apple教學
文章字數： 814 所需閱讀時間 ≈ 1 分鐘

很多 Mac 用戶在清理硬碟空間時，會發現 Library 電腦資料庫的 Upadate 吃了好多容量，這資料能不能刪除？這篇告訴你 MacOS Upadte 資料夾是什麼，以及正確的移除方式，幫你清理出儲存空間 😆

Mac 的 Update 資料夾是什麼？

資料庫裡的 Update 是來_存放 MacOS 的更新檔案_，因此會佔不少空間，動輒就吃掉硬碟好幾 GB。裡面的更新檔是可以移除的，但有些特定檔案要留下來，否則會影響系統運作喔。

如果你的 Mac 硬碟空間快爆滿，可以清理 Update 裡的檔案來壓出一些空間～

總結

總之來說，Mac 電腦的 Update 大多更新檔事可以刪除的，只是要注意不要刪到特定的系統文件。如果你的 Mac 硬碟快爆掉了，可以重開機，讓系統清理緩存，也能清理一些儲存空間 ～
